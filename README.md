## Connected Gallery Project
A modern, interactive gallery showcase built with HTML, CSS, and Vanilla JavaScript, featuring three unique gallery experiences combined into a single responsive web page.

## 🚀 Features
1. Flow Gallery
An immersive carousel with:
Infinite circular navigation
Auto-play functionality
Thumbnail navigation
Swipe support for touch devices
Smooth animated transitions
Responsive layout

2. Tilted Carousel
A stylish card-based gallery featuring:
3D tilted card effect
Previous/Next navigation
Interactive dot indicators
Dynamic location captions
Touch swipe support
Mobile-friendly design

3. Circular Gallery
A rotating wheel-style image gallery with:
Circular landmark navigation
Animated rotating ring
Large central preview lens
Auto-rotation
Keyboard accessibility
Touch gesture support
Smooth image transitions
